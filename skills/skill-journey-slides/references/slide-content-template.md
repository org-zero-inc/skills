# 幻灯片内容模板

本文件提供每个阶段幻灯片的内容模板，供生成演示文稿时参考。

## Title Slide

```
标题：[Skill 名称] — 编写旅程
副标题：从发现问题到验证完成
底部：生成日期 | 五阶段回顾
```

## Overview Slide — 旅程路线图

```
标题：旅程概览

可视化五阶段流程：
🔍 发现问题 → 🔍 分析问题 → 📄 Skill 计划 → 🔧 Skill 调整 → 🧪 Skill 验证

每个阶段下方一行简短描述（8-12字）
```

## Stage 1 — 发现问题（2 slides）

### Slide 1：初始需求

```
标题：🔍 发现问题 — 初始需求

要点：
- 用户原始请求：[引用用户原话]
- 当前痛点：[1-2 句描述]
- 为什么需要这个 Skill：[1-2 句描述]
```

### Slide 2：问题本质

```
标题：🔍 发现问题 — 问题本质

要点：
- 核心问题：[一句话概括]
- 影响范围：[谁受影响、什么场景]
- 期望效果：[理想状态是什么]
```

## Stage 2 — 分析问题（2 slides）

### Slide 1：需求拆解

```
标题：🔍 分析问题 — 需求拆解

要点：
- 功能点 1：[描述]
- 功能点 2：[描述]
- 功能点 3：[描述]
- 关键约束：[技术/环境/用户偏好约束]
```

### Slide 2：方案思考

```
标题：🔍 分析问题 — 方案思考

要点：
- 方案 A：[描述] — 优点 / 缺点
- 方案 B：[描述] — 优点 / 缺点
- 最终选择：[方案] — 原因
```

## Stage 3 — Skill 计划（2 slides）

### Slide 1：架构设计

```

- 触发条件：[什么场景触发]
- 输入/输出：[输入什么、输出什么]
```

### Slide 2：依赖与协作

```
标题：📄 Skill 计划 — 依赖与协作

要点：
- 核心 Skill：[依赖的 Skill 名称]

- 关键设计决策：[1-2 个重要选择及原因]
```
